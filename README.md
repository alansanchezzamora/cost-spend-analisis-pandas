# Overview

The **Cost Spend Analisis tool** is an application designed to analize a dataset for cloud spend.  
My goal in this project is to deepen my understanding of **PANDAS**.  

This project uses matplotlib for data visualization.

[Software Demo Video](https://youtu.be/viwb0mpXfYs)

**Technologies and Tools Used:**
- Data filtering
- Aggregation (sum, average)
- Sorting
- Date conversion
- Data visualization using Matplotlib
